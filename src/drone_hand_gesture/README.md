## 🔥 核心亮点

### 1. 基于 OpenCV 肤色检测的手势识别（无 MediaPipe 依赖）

纯 OpenCV 实现，通过 **HSV + YCrCb 双色彩空间肤色检测**和**轮廓分析**，在无 MediaPipe 环境下也能实现高精度手势识别。输出 21 个手部关键点，与 MediaPipe 格式完全兼容。

```
┌─────────────────────────────────────────────────┐
│              摄像头图像输入                        │
│                    ↓                              │
│          HSV 肤色检测 + YCrCb 肤色检测              │
│                    ↓                              │
│         形态学处理 → 轮廓提取 → 凸包分析             │
│                    ↓                              │
│      手指角度计算 + 手掌张开程度量化                  │
│                    ↓                              │
│         21 关键点映射 → 手势分类 → 控制指令          │
└─────────────────────────────────────────────────┘
```

### 2. 手势强度分级控制

手指角度与手掌张开程度联动，实现**无级调速**。手掌张开越大，无人机速度越快；手指并拢越多，速度越慢。强度系数 $I \in [0, 1]$ 实时计算：

- **手掌张开程度**：通过凸包面积与轮廓面积之比计算
- **手指角度**：指尖到指根连线与手掌平面夹角
- **最终强度**：$I = \alpha \cdot \text{palm\_openness} + \beta \cdot \text{finger\_angle}$

### 3. 双手控制模式

| 左手（方向控制） | 右手（高度控制） |
|:---:|:---:|
| 胜利手势 → 前进 | 食指向上 → 上升 |
| 拇指向上 → 后退 | 食指向下 → 下降 |
| 食指向左 → 左转 | OK 手势 → 悬停 |
| 食指向右 → 右转 | — |
| 滑动手势 → 平移 | — |

**双手通用命令**：张开手掌（起飞）、握拳（降落）、拇指向下（急停）

### 4. 多无人机双手协同控制

支持同时操控 2 架无人机，左手控制 Drone 1 的方向，右手控制 Drone 2 的高度。支持三种控制模式：

| 模式 | 说明 |
|:---:|------|
| `dual` | 双手分离：左手→D1，右手→D2 |
| `single` | 单机控制：仅操作选中的无人机 |
| `sync` | 同步控制：双手操控所有无人机 |

支持 **V字形、直线、方形、三角形** 四种编队飞行。

## 🚀 快速开始

### 环境要求

- **Python** ≥ 3.8
- **摄像头**：USB / 内置摄像头
- **系统**：Windows / macOS / Linux

### 安装

```bash
# 进入项目目录
cd src/drone_hand_gesture

# 安装基础依赖（OpenCV 肤色检测模式，无需 MediaPipe）
pip install -r requirements.txt
```

### 运行

```bash
# 方式一：GUI 启动器（推荐）
python launcher.py

# 方式二：命令行直接启动
python main.py

# 方式三：AirSim 仿真模式（需先安装 AirSim）
python main_airsim.py

# 方式四：深度学习模式
python main_deep_learning.py
```

## 🖐️ 手势识别模式

系统支持**三种手势识别模式**，按优先级自动选择：

| 优先级 | 模式 | 依赖 | 精度 | 说明 |
|:---:|------|------|:---:|------|
| 1 | **OpenCV 肤色检测** | 仅 OpenCV | ★★★☆ | 零额外依赖，开箱即用 |
| 2 | **MediaPipe 规则检测** | MediaPipe | ★★★★ | 21 关键点高精度追踪 |
| 3 | **机器学习分类** | MediaPipe + sklearn | ★★★★★ | 训练后精度最高 |

系统启动时会**自动检测可用模式**并选择最优方案。如果没有安装 MediaPipe，会自动降级到 OpenCV 肤色检测模式。

### OpenCV 肤色检测技术细节

```
技术栈：
  ├── HSV 色彩空间肤色范围
  │   └── H: [0, 20]  S: [20, 255]  V: [70, 255]
  ├── YCrCb 色彩空间辅助检测
  │   └── Cr: [133, 173]  Cb: [77, 127]
  ├── 形态学操作（开运算 + 闭运算）
  ├── 轮廓查找与凸包分析
  ├── 凸缺陷检测 → 指尖定位
  ├── 21 关键点模拟生成（兼容 MediaPipe 格式）
  └── 手势规则匹配 + 历史平滑滤波
```

## 🎚️ 手势强度分级控制

手势强度通过**手指角度**和**手掌张开程度**两个维度综合计算，实现无人机的无级调速。

### 强度计算模型

```
强度 I = w₁ × palm_openness + w₂ × finger_angle_ratio

其中：
  palm_openness   = 凸包面积 / 轮廓面积（0~1）
  finger_angle    = 手指伸直程度归一化（0~1）
  w₁, w₂          = 权重系数（默认各 0.5）
```

### 强度分级效果

| 强度范围 | 等级 | 无人机响应 | 视觉效果 |
|:---:|:---:|------|:---:|
| 0.0 ~ 0.3 | 低 | 缓慢移动 | 手掌微张 |
| 0.3 ~ 0.6 | 中 | 正常速度 | 手掌半开 |
| 0.6 ~ 0.8 | 高 | 快速移动 | 手掌张开 |
| 0.8 ~ 1.0 | 极速 | 最大速度 | 手指完全伸展 |

> 强度信息会实时显示在控制面板的 `INTENSITY` 指示条上。

## 🎛️ 动态灵敏度调节

灵敏度调节影响手势识别的**置信度阈值**和**触发灵敏度**，三档可调。

### 灵敏度档位

| 档位 | 键位 | 手势阈值 | 滑动阈值 | 适用场景 |
|:---:|:---:|:---:|:---:|------|
| **LOW** 🔴 | `[` | 0.75 | 0.30 | 复杂背景，减少误识别 |
| **MEDIUM** 🟡 | `=` | 0.60 | 0.20 | 日常使用，推荐默认值 |
| **HIGH** 🟢 | `]` | 0.45 | 0.12 | 简单背景，提高响应速度 |

### 调节方式

```
键盘调节：
  [  → 降低灵敏度（更严格）
  ]  → 提高灵敏度（更宽松）
  =  → 重置为 MEDIUM

运行时实时生效，无需重启
```

> 当前灵敏度状态会显示在画面右上角的 `SENSITIVITY` 指示器上。

### 编队飞行

支持四种预设编队，队长跟随手势移动，队员保持相对位置：

| 编队 | 形状 | 适用场景 |
|:---:|:---:|------|
| `line` | ═══ 直线 | 列队巡航 |
| `v_shape` | ∧ V字形 | 侦察搜索 |
| `square` | □ 方形 | 区域覆盖 |
| `triangle` | △ 三角形 | 突击编队 |

### 操作方式

```python
# 代码中切换编队
formation.set_formation("v_shape")   # 切换为 V 字形

# 编队移动
formation.move_formation("forward", intensity=0.6)
```

```bash
# 启动数据采集工具
python gesture_data_collector.py
```

采集工具会自动拍摄手势图像并保存到 `dataset/` 目录，支持自定义手势标签。

### 训练传统 ML 模型

```bash
# 训练 SVM / 随机森林 / MLP
python train_gesture_model.py --model svm
```

### 训练深度学习模型

```bash
# 训练 CNN
python train_deep_gesture.py --model_type cnn --epochs 100

# 对比训练所有模型
python train_deep_gesture.py --compare

# 评估模型
python compare_algorithms.py
```
